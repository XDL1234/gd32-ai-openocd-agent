## 快速查找指南

### 按开发阶段查找

**RESEARCH 阶段开始时**：
1. 阅读 SKILL.md "快速工具索引"查找搜索工具
2. 阅读 "容灾备份与降级策略"了解工具降级方案
3. 执行引脚规划时参考 `refs/pin-planning.md`

**编写代码时**：
1. 参考 `refs/coding-standards.md` 确保代码规范
2. 使用 IMU 时参考 `refs/imu-gyroscope-checklist.md` 和 `refs/mahony-ahrs-reference.md`
3. 遇到问题时参考 `refs/troubleshooting.md`

**需要移植驱动时**：
1. 参考 `refs/driver-porting.md` 了解移植流程
2. 进行跨平台时参考 `refs/platform-migration.md`
